# clear-skills

Clear AI skills — remove skill files from the current project or globally

```bash
npx clear-skills
```

### Options

| Option                    | Description                                                                                                                                        |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| `-g, --global`            | Scan globally instead of in the current project                                                                                                       |
